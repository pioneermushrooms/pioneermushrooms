# Hey, I'm Chris

I build AI-powered systems by wiring together APIs, models, and services into products that work end-to-end. My focus is on **integration architecture** — connecting the right tools so they do real work autonomously.

---

## What I Build

### MAIGENT — Self-Evolving AI Agent
A multi-crew AI agent deployed as a Discord bot, powered by CrewAI. Routes tasks across specialized crews (research, data, comms, code, ops) with automatic LLM selection. Builds its own tools, runs its own research, and manages itself.

### TWDT Video Bot — Automated Match Recaps
Generates weekly recap videos for the Trench Wars Draft Tournament. Scrapes forum posts, generates narration scripts, overlays a talking-head avatar onto game footage, and produces polished 1080p videos — all from a single command.
